# Mini ERP PHP

Sistema de gerenciamento de produtos e pedidos desenvolvido em PHP com arquitetura MVC.

## Funcionalidades

- Cadastro de produtos
- Carrinho com sessão
- Aplicação de cupom de desconto
- Finalização de pedido
- Redução automática de estoque
- Persistência em MySQL

## Tecnologias

- PHP 8+
- MySQL
- PDO
- Bootstrap
- Arquitetura MVC

## Como rodar

1. Clone o projeto
2. Importe o banco de dados
3. Configure `configuração/db.php`
4. Rode via XAMPP ou WAMP
5. Acesse `http://localhost/Mini-ERP-PHP/publico`
